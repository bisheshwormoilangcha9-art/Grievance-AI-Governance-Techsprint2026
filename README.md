# GrievanceSense - AI Citizen Grievance Intelligence Platform

GrievanceSense is an AI-powered system designed to analyze, prioritize, and manage citizen grievances related to public services such as water, electricity, roads, healthcare, and sanitation.

## Problem Statement
Traditional grievance systems only collect complaints without intelligent analysis. This leads to delays, poor prioritization, and lack of data-driven insights.

## Our Solution
GrievanceSense uses Artificial Intelligence and Natural Language Processing to:
- Categorize grievances automatically  
- Detect urgency and credibility  
- Identify complaint hotspots  
- Support faster government response  

## Tech Stack
- Python  
- Scikit-learn  
- Pandas  
- Streamlit  
- TF-IDF  
- Pickle
